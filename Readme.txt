# Portfolio d'Ulrich Waba

## Description
Bienvenue sur mon portfolio ! Je suis Ulrich Waba, un développeur passionné avec une expertise en C++, SQL, développement web avec Angular, et projets matériels avec Arduino. Ce portfolio présente mes projets, compétences et expériences professionnelles.

## Projets
### 1. Logiciel de Gestion de Stock
- **Technologies** : C++, SQL
- **Description** : Développement d'un logiciel de gestion de stock pour une entreprise, avec une interface utilisateur intuitive et des fonctionnalités de gestion des produits, des fournisseurs et des commandes.

### 2. Voiture Télécommandée par Bluetooth
- **Technologies** : Arduino, Module Bluetooth HC-05
- **Description** : Création d'une voiture télécommandée via Bluetooth, contrôlée par une application mobile. Projet intégrant des compétences en électronique et programmation embarquée.

### 3. Site de Vente de Pièces en Ligne
- **Technologies** : Angular, API REST
- **Description** : Développement d'un site de vente de pièces automobiles en ligne, avec une interface utilisateur dynamique et une gestion efficace des données et des transactions.

## Compétences
- **Langages de Programmation** : C++, JavaScript, TypeScript, SQL
- **Frameworks et Bibliothèques** : Angular, Node.js, Express, Qt
- **Outils et Technologies** : Git, GitHub, Arduino, MySQL, MongoDB
